# Actividad Integradora
## Arreglos Unidimensionales

## Objetivo de proyecto
Desarrollar una aplicación web básica para el registro y procesamiento estadístico de una lista de productos, calculando totales, promedios y valores extremos (máximos y mínimos).

## Problemas que resuelve
Resuelve la necesidad de automatizar el cálculo de métricas financieras básicas sobre un lote de productos ingresados por un usuario, evitando errores manuales al determinar qué artículo tiene el precio más alto o más bajo.

## Tecnologías utilizadas
- PHP
- HTML
- CSS

## Conceptos aplicados
- Expresiones Aritméticas
- Arreglos en java
- Constructor array()
- Arreglos asociativos y indexados
- La funcion print_r()

## Instrucciones de ejecución
- Coloca los archivos index.php, procesar.php y resultados.php dentro de la carpeta htdocs de tu servidor local (XAMPP).
- Inicia los servicios de Apache en el panel de control de XAMPP.
- Abre tu navegador y accede 127.0.0.1.
- Completa el formulario con los datos de los 5 productos y presiona "Procesar" para ver los resultados.

## Reflexión personal

Aprendí a utilizar arreglos en formularios HTML para enviar múltiples entradas a la vez y cómo pasarlos entre diferentes scripts PHP de forma encadenada.
